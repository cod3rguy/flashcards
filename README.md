# Flashcards

Application to build Flashcard Decks, built using React.

Hosted at: http://lab.coderguy.tech/flashcards

### Technologies Used

- html
- css
- less
- bootstrap
- javascript es2015
- react
- babel
- browser-sync
- gulp

### How to

- `git clone <url>`
- `npm install`
- `npm start`

To remove the example data (JavaScript deck), edit `/scripts/src/includes/data-config.js`.
